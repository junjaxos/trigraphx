# TriGraphX Streamlit UI - 数据可视化界面

一个交互式的 Web UI 仪表板，用于可视化、查询和管理 TriGraphX 数据库。

## 🎯 功能

### 📊 总览面板
- **实体统计** - 实体总数、查询次数、存储大小
- **类型分布** - 度量类型的饼图展示  
- **存储分析** - 实体文件、索引、检查点、日志存储分布
- **系统配置** - 最大实体数、版本信息

### 🔍 交互式查询
- **KNN查询** - 找K个最近邻，支持度量类型选择
- **范围查询** - 指定搜索半径找所有实体
- **路径查询** - 找两个实体间的最短路径
- **多指标查询** - 自定义权重的加权组合查询

每次查询显示：
- 查询结果表格
- 距离/相似度可视化
- 查询耗时

### 📈 关系可视化
- **网络拓扑** - 使用 PyVis 展示实体关系网络（交互式）
- **关联矩阵** - 实体相似度热力图
- **向量空间** - t-SNE 投影的 2D 向量空间

### ⏱️ 性能监控
- **查询延迟统计** - 平均/最大/最小延迟
- **延迟趋势** - 时间序列图
- **延迟分布** - 直方图展示
- **查询类型占比** - 各类型查询的统计

### 💾 数据管理
- **添加实体** - 支持 JSON 数据和多种嵌入类型
- **查看实体** - 列表展示所有实体
- **删除实体** - 支持软删除（可恢复）和硬删除
- **备份恢复** - 快照创建和检查点恢复

### ℹ️ 帮助文档
- TriGraphX 概念说明
- 4种度量类型介绍
- 4种查询类型说明
- 快速代码示例
- FAQ

## 📦 安装依赖

```bash
# 安装所有依赖
pip install -r requirements.txt

# 或只安装 UI 必需的
pip install streamlit plotly pandas pyvis
```

## 🚀 启动 UI

```bash
# 方式 1: 直接运行
streamlit run ui_streamlit.py

# 方式 2: 指定端口
streamlit run ui_streamlit.py --server.port 8080

# 方式 3: 指定地址和端口
streamlit run ui_streamlit.py --server.address 0.0.0.0 --server.port 8080
```

然后在浏览器中打开：
- 本地: http://localhost:8501
- 远程: http://<server_ip>:8501

## 📖 使用指南

### 1. 开始使用

启动后会看到左侧导航栏，选择不同功能：

```
📊 总览      → 查看数据统计
🔍 查询      → 执行各种查询
📈 可视化    → 查看关系图
⏱️ 性能      → 监控查询性能
💾 数据管理  → 添加/删除/备份数据
ℹ️ 帮助      → 查看使用说明
```

### 2. 添加测试数据

**方式1: 通过 UI**
1. 进入 💾 数据管理 → 添加实体
2. 填写实体ID、JSON数据
3. 选择嵌入类型（语义、层级、关联、因果）
4. 点击"添加实体"

**方式2: 通过代码**
```python
# 先运行示例来加载测试数据
python3 examples.py
```

### 3. 执行查询

以 KNN 查询为例：

1. 进入 🔍 查询 → KNN查询
2. 选择源实体 ID
3. 设置 K 值（通常 3-10）
4. 选择度量类型
5. 点击 🚀 执行查询
6. 查看结果表格和可视化

### 4. 查看可视化

- **网络拓扑**: 展示实体间的关联关系（限制20个实体以保证性能）
- **关联矩阵**: 实体对之间的相似度（热力图，越蓝越相似）
- **向量空间**: 高维语义向量投影到 2D 平面

### 5. 监控性能

- 每次查询会记录耗时
- 📊 总览 面板实时显示查询次数
- ⏱️ 性能 面板显示查询历史和趋势

### 6. 管理数据

- **添加**: JSON 格式数据 + 嵌入类型
- **查看**: 表格展示所有实体
- **删除**: 软删除（可恢复）或硬删除（永久）
- **备份**: 创建快照或从快照恢复

## 🎨 界面特色

- 📱 **响应式设计** - 自动适配桌面和平板
- 🎨 **现代配色** - 梯度蓝紫配色方案

## 🔧 配置

### 修改端口

```bash
streamlit run ui_streamlit.py --server.port 9000
```

### 修改主题

在浏览器访问 UI 后，右上角菜单 → Settings → 选择主题

### 配置数据目录

编辑 `ui_streamlit.py`，修改:
```python
db_root = Path("trigraphx_data")  # 改为你的路径
```

## 🐛 常见问题

### Q: 启动时提示 "Missing module"?
A: 安装缺失的依赖: `pip install streamlit plotly pandas pyvis`

### Q: PyVis 网络图显示空白?
A: 需要安装 pyvis: `pip install pyvis`

### Q: 查询很慢?
A: 可能实体太多。检查 ⏱️ 性能 面板，调整 K 值或缩小范围

### Q: 数据丢失了?
A: 检查 💾 数据管理 中的备份恢复，或查看 `trigraphx_data/` 目录

### Q: 如何导出数据?
A: 进入 💾 数据管理 → 查看实体，右上角有下载选项
